## The Powerlifter's Equipment Hierarchy

A powerlifter's gear priorities are different from general fitness:

1. **Power rack with safety bars** (squat + bench)
2. **Olympic barbell** (the bar matters more than the plates)
3. **Plates** (cast iron is fine — no need for bumpers)
4. **Adjustable bench** (flat for bench press, can angle for accessories)
5. **Lifting belt** (mandatory for max effort)
6. **Trap bar** (back-friendly deadlift variation for accessory work)
7. **Roman chair** (lower back injury prevention)

This build covers all 7. No fluff.

## What's Missing

The $1,500 powerlifter build doesn't include:
- **Cardio equipment** (powerlifters don't need much cardio, but a $15 jump rope helps)
- **Adjustable dumbbells** (helpful for accessories, $150-430 to add later)
- **Bumper plates** (only if you do drop deadlifts, which most powerlifters don't)
- **Specialty bars** (safety squat bar, cambered bar — premium upgrades)

These are nice-to-haves, not must-haves. Add them as you progress.

## The Powerlifter's 12-Week Beginner Program

This program uses only the build above:

### Day 1: Squat Focus
- Back Squat: 5x5 (work up to a heavy 5)
- Romanian Deadlift: 3x8
- Bulgarian Split Squat: 3x10 each leg
- Hyperextensions: 3x12

### Day 2: Bench Focus
- Bench Press: 5x5
- Close-Grip Bench: 3x8
- Bent-Over Row: 4x6
- Pull-Ups: 3x to failure

### Day 3: Deadlift Focus
- Deadlift: 3x5 (work up to a heavy 3-5)
- Trap Bar Deadlift: 3x8 (lighter than main)
- Hyperextensions: 3x15
- Plank: 3x60 sec

Add weight to all main lifts when you complete all sets. This is the foundation of powerlifting strength.
