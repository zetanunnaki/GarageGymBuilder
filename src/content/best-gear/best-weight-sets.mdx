## Cast Iron vs Bumper Plates

### Cast Iron (What Most Sets Include)
- Cheaper per pound
- Thinner — more plates fit on the bar
- LOUD when dropped
- Can crack concrete if dropped without mats

### Bumper Plates (Sold Separately)
- Safe to drop from overhead
- Required for Olympic lifts (clean, snatch)
- Thicker — fewer plates fit on the bar
- More expensive per pound

**Our recommendation:** Start with the cast iron set, add a pair of bumper 45s later if you do Olympic lifts or deadlifts with drops.

## When to Buy Separately

Buy a set if you need everything. Buy separately if:
- You already have a quality barbell
- You want specific plate types (bumpers, calibrated, competition)
- You need more than 300 lbs total
